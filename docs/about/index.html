Work

Hi, my name is Guilherme Leobas. I am currently working at @quansightai as a Software Engineer. Before joining Quansight, I was a MSc student at the Federal University of Minas Gerais (dcc/ufmg) working with compiler optimizations under the supervision of professor Fernando Magno.

As part of my work at Quansight, I have contributed to a number of open source softwares:

  • Numba, a NumPy aware JIT compiler using LLVM
  • RBC, a remote JIT compiler built on top of Numba
  • OmniSciDB, one of the fastest SQL engine in the world
  • PyTorch, an open source ML framework
